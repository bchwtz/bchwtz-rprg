Aufgabe 4

Eine binäre Folge der Länge \(n\) ist eine endliche Folge \(s = \left(s_1, s_2, ..., s_n\right)\) mit \(s_j \in \{-1, +1\}\) für alle \(1 \leq j \leq n\). Für \(0 < k < n\) ist die aperiodische Autokorrelation (der Ordnung k) definiert durch:

\[C_k(s) = \sum_{j=1}^{n-k} s_j \cdot s_{j+k}.\]

In R kann \(s\) leicht durch einen Integer-Vektor dargestellt und die aperiodische Autokorrelation bei-spielsweise wie folgt implementiert werden:

autokorrelation1 <- function(s) {
  n <- length(s)
  res <- vector(mode = "integer", length = n - 1)
  for (k in 1:(n-1)) {
    res[k] <- as.integer(t(s[1:(n - k)]) %*% s[(k + 1):n])
  }
  return(res)
}

  1. Modifizieren Sie die Funktion autokorrelation1(), indem Sie die Schleife durch map_int() (aus purrr (Henry and Wickham 2020)) ersetzen. Nennen Sie die modifizierte Version autokorrelation2().

  2. Vergleichen Sie die Laufzeiten der beiden Funktionen für große n mithilfe von system.time(), beispielsweise durch:

    randBinSeq <- function(n) {
  return(sample(c(-1L, 1L), n, replace = TRUE))
}
s <- randBinSeq(2E4) 
autokorrelation1(s)
autokorrelation2(s)

  3. Erstellen Sie einen weiteren Laufzeittest; erklären und interpretieren Sie die Ergebnisse:

    require(compiler)
f <- cmpfun(autokorrelation1)
system.time(f(s))
