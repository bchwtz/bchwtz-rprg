Der Datensatz diamonds ist ein „Tibble“, der annähernd 54,000 Diamanten beschreibt. Jede dieser Observationen besteht aus 10 Variablen, die unter anderem Informationen über Preis, Karat, Farbe, Reinheit und Schliff enthalten:

Variable dt. Übersetzung Bedeutung
price Preis price in US dollars ($326–$18,823)
carat Karatgewicht weight of the diamond (0.2–5.01)
cut Schliff quality of the cut (Fair, Good, Very Good, Premium, Ideal)
color Farbe diamond colour, from D (best) to J (worst)
clarity Reinheit a measurement of how clear the diamond is (I1 (worst), SI2, SI1, VS2, VS1, VVS2, VVS1, IF (best))
x length in mm (0–10.74)
y width in mm (0–58.9)
z depth in mm (0–31.8)
depth total depth percentage = z / mean(x, y) = 2 * z / (x + y) (43–79)
table width of top of diamond relative to widest point (43–95)

diamonds %>% count()

    diamonds %>% group_by(carat) %>% summarize(n=n())
diamonds %>% group_by(carat) %>% tally()
diamonds %>% group_by(carat) %>% count()

  3. Wie kann man mithilfe von count() oder tally() Folgendes kürzer schreiben?

    diamonds %>% group_by(carat) %>% summarize(n = n())

  4. Wie kann man mithilfe von count() oder tally() Folgendes kürzer schreiben?

    diamonds %>% summarize(n = sum(carat))

Aufgabe 4

  1. Was gibt die Funktion ganzOderWeniger() zurück?

    ganzOderWeniger <- function(x, delta) {
  return(floor(x + delta) == ceiling(x))
}

  2. Erklären Sie den folgenden Quelltext und werten Sie die entsprechende Ausgabe aus. Was ist daran auffällig?

    diamonds %>% 
  filter(ganzOderWeniger(carat, 0.05)) %>%  
  count(carat)

# äqivalent hierzu:
diamonds %>% 
  filter(ganzOderWeniger(carat, 0.05)) %>% 
  group_by(carat) %>%
  summarize(n = n())

  3. Vergleichen Sie folgende Variante mit Aufgabenteil b)? Gibt es die Auffälligkeiten immer noch?

    diamonds %>%

  count(carat) %>% 
  print(n = Inf)
